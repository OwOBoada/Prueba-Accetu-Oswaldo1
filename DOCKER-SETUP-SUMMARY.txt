🚀 INICIO RÁPIDO:

OPCIÓN 1: Windows - Ejecución Local
  1. Abre PowerShell o CMD
  2. Ve a: c:\Users\owo\Music\Prueba Accetu Oswaldo
  3. Ejecuta: run-docker.bat
  4. Accede a: http://localhost:8080/api/franchises

OPCIÓN 2: Linux/Mac - Ejecución Local
  1. cd ~/tu-ruta/franchiseapi
  2. docker-compose up -d
  3. curl http://localhost:8080/api/franchises

OPCIÓN 3: AWS EC2 - Despliegue Automatizado
  1. Crea una instancia EC2 (Ubuntu 22.04)
  2. Conéctate por SSH
  3. curl -fsSL https://tu-repo/deploy-aws-ec2.sh | bash
  4. Accede a: http://tu-ip-ec2:8080/api/franchises

🔧 REQUISITOS:

VERSIONES:
  • Docker: 20.10+ (con Docker Compose)
  • Java: 21 (incluido en Dockerfile)
  • Maven: 3.9.14 (usado en build)
  • MySQL: 8.4 (en container)

HARDWARE (recomendado):
  • Local: 4GB RAM, 10GB disco
  • AWS: t3.small (2GB RAM, 20GB disco)
