# 06 - Recursos y cuentas necesarias

## Obligatorio para esta entrega

- Cuenta GitHub (repositorio público)
- Git instalado
- Docker + Docker Compose
- JDK 21 (si no se usa Docker para build)
- Maven 3.9+ (si no se usa Docker para build)

## No obligatorio para esta entrega

- Cuenta AWS
- Cuenta de base de datos administrada en la nube
- Terraform/CloudFormation

## Opcional recomendado (siguiente fase)

- Cuenta AWS para despliegue real
- Docker Hub o Amazon ECR para publicar imagen
- Terraform para aprovisionamiento repetible
